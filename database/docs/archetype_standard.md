# Archetype Standard

## Runtime source of truth

- Authored seed markdown: `/database/archetypes/ARCHETYPE_XX_seed.md`
- Generated archetype JSON: `/database/archetype_models/ARCHETYPE_XX.json`

`/database/docs/archetypes/**` is documentation-only.

## Required contract

Seed files must follow `/database/docs/archetypes/archetype_seed_template.md` and `/database/docs/archetypes/archetype_seed_schema.md`.
