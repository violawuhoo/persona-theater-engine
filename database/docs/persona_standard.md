# Persona Standard

A persona is a generated artifact.

## Inputs and outputs

- Input seed: `/database/archetypes/ARCHETYPE_XX_seed.md`
- Generated persona JSON: `/database/personas/ARCHXX.json`
- Generated archetype JSON: `/database/archetype_models/ARCHETYPE_XX.json`

## Policy and schema references

- Policy: `/database/docs/personas/persona_generation_policy.md`
- Schema: `/database/schema/persona.schema.json`
