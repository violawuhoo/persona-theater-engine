# Persona Draft: Identity Collapse Engine

An advanced behavioral rehearsal tool designed to bridge the gap between AI-driven persona logic and physical human execution. 

## 🚀 Overview
The **Persona Draft** engine is a "social biohacking" interface. It doesn't just chat; it recalibrates the user's physiological and cognitive state to match high-stakes, logic-driven personas (e.g., the ARCH-01 "Queen" protocol).

## 🛠️ Core Features
- **Dynamic Persona Loading**: Automatically indexes and loads structured JSON protocols from the `/database/personas/` directory.
- **Identity Collapse Sequence**: A 5-second timed ritual to synchronize breathing, gaze, and posture before social entry.
- **Theater Mode HUD**: A real-time dashboard providing "Mind-Body-Speech" instructions based on specific interaction scenarios.
- **Resilient Logic Engine**: Built-in fault tolerance for inconsistent data formats and API latency.

## 📂 Project Structure
- `/database/personas/`: Contains `.json` shadow files (executable logic) and `.md` source files (persona DNA).
- `index.html`: The central interface (The Wheel & Theater).
- `script.js`: The "Tactical Engine" handling data fetching, normalization, and API calls.
- `style.css`: The visual "Atmosphere" layer, including the breathing metronome and transition effects.

## ⚙️ Setup & Installation
1. **Clone the repo**: `git clone https://github.com/YOUR_USERNAME/persona-draft.git`
2. **Local Server**: Due to CORS restrictions when fetching local JSON, use a local server.
   - If using VS Code, install **Live Server** and click "Go Live".
3. **Environment Variables**: Add your `GEMINI_API_KEY` to your environment to enable real-time tactical generation.
